fonction qui affiche le menu pour un joueur afin de choisir la colonne ou il souhaite jouer le pion. 

cette fonction retourne le numéro de la colonne (entier compris entre 1 et \-M) choisis par le joueur. retourne 0 si le joueur choisis de revenir au menu précédant pour changer de pion. 
\begin{DoxyParams}{\-Paramètres}
{\em nom} & le nom du joueur pour l'afficher dans le menu ou dans les messages d'erreurs. \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{\-Renvoie}
le numéro de la colonne choisis (0 pour changer de pion / entre 1 et 7 pour placer le pion dans une colonne). 
\end{DoxyReturn}
